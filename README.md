# Ders Kayıt Sistemi

Bu proje, C# kullanarak katmanlı mimari ile geliştirilmiş bir ders kayıt sistemidir. Sistem, öğrencilerin mevcut derslere kayıt olmasını ve başvurularını görüntülemesini sağlar. Projede üç ana sayfa bulunmaktadır: Öğrenciler, Açılan Dersler ve Başvurular.

## Özellikler

- Açılan derslerin listelenmesi ve ders kaydı.
- Seçilen dersler için başvuru yapılması.
- Temel CRUD işlemleri ile veri yönetimi.

## Kullanılan Teknolojiler

- **ASP.NET Core MVC**: Web arayüzü oluşturmak için kullanıldı.
- **Entity Framework Core**: Veri erişimi ve yönetimi için kullanıldı.
- **SQL Server**: Veritabanı yönetimi için kullanıldı.
